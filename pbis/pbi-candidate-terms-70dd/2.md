## Overview

Add a new diagnostic query ŌĆö Q8 ŌĆö to `peddlerr/workbench/docs/operations/diagnostics/semantic_config_inventory.sql`. Q8 partitions Phase 2's undermapped term set into the six candidate-term lifecycle states and surfaces prior `evaluation_notes` for already-evaluated candidates. Q8 is the core artifact that lets `reconcile-canonical-language` operate over the accumulated candidate set rather than re-evaluating from scratch.

### What if we didn't do this step?

**Immediate Impact:** Even with the schema and recording in place, Phase 2 has no read-side query that joins undermapped terms to prior decisions. The accumulated state would exist but be invisible to the skill.

**Downstream Consequences:** Task 6 (skill update) cannot point at a query, so prior rationale never reaches the agent.

**Technical Debt:** Q7 remains the only diagnostic, and it has no awareness of prior evaluations.

**Alternative Approaches:** The skill could issue ad-hoc SQL inline, but a named diagnostic in the canonical inventory file is the project convention.

**Risk Assessment:** Low ŌĆö read-only diagnostic.

## Description

Q8 produces one row per undermapped term with the following columns:

- `vocabulary_term_id`
- `normalized_label` (or whatever the corresponding column is on `vocabulary_terms`)
- `source_system`
- `surface_id` / `surface_name`
- `bucket` ŌĆö one of `never_seen`, `candidate`, `deferred`, `merged`, `rejected`, `promoted`
- `candidate_term_id` (nullable for `never_seen`)
- `prior_evaluation_notes` (nullable; from `candidate_terms.evaluation_notes`)
- `cross_source_count` ŌĆö count of distinct `source_system` values across observations of the same candidate (1 for `never_seen`)
- `merged_into_candidate_id` (nullable; populated only for `merged`)
- `resolved_concept_key` (nullable; populated only for `promoted`)

The `promoted` bucket appearing in the undermapped set is itself a flag ŌĆö a term that's promoted should already have a mapping, so its presence in the undermapped set indicates a mapping bug worth surfacing.
